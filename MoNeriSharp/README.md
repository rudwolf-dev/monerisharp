# moNeriSharp

Proyecto de IA modular con TorchSharp y datasets multilinguees y tematicos.

## Estructura
- src/: codigo C# de modulos y pipeline
- data/: datos raw/clean/splits
- configs/: hiperparametros y rutas
- notebooks/: exploracion
- docs/: documentacion (incluye Obsidian
- tests/: pruebas unitarias
- logs/: resultados de entrenamiento
- scripts/: utilidades

## Notas
Ver docs/obsidian para la documentacion detallada por dataset y modulo.
